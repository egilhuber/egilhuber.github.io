Dice Game

Final Product

Goals of this project

Game must:
-be played with dice and more complicated than war
-have gameplay functionality
-have one function to roll dice, regardless of sides
-utilize 6 different polyhedral dice
-make consistent commits to repository with good, detailed messages
-have two player capability
-have a responsive button to roll dice
-continue until game is won or lost
-have an automated bonus round
-allow player to select die for scoring
-have visuals for dice
-have an interactive front end with responsive score card
